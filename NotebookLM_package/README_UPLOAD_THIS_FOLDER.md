# NotebookLM 上傳說明

這個資料夾是專門給 NotebookLM 使用的資料包。

## 建議上傳到 NotebookLM 的檔案

請把整個 `NotebookLM_package` 資料夾中的檔案都上傳到 NotebookLM，或至少上傳以下檔案：

```text
NOTEBOOKLM_PROMPT.md
SOURCE_LINKS.md
PROJECT_CONTEXT.md
MODEL_RESULTS.md
REPORT_DRAFT.md
ASSIGNMENT_REQUIREMENTS.md
README_FOR_PROJECT.md
model_metrics.csv
selected_features.csv
```

## 使用方法

1. 開啟 NotebookLM。
2. 建立新的 notebook。
3. 上傳本資料夾內的檔案。
4. 依照 `SOURCE_LINKS.md` 加入 Kaggle、UCI 與 scikit-learn 來源連結。
5. 打開 `NOTEBOOKLM_PROMPT.md`。
6. 將整份 prompt 複製貼到 NotebookLM 的對話框。
7. 取得 NotebookLM 產出的「NotebookLM 摘要」與「網路主流或更優解法比較」。
8. 將結果貼回正式報告。

## 最重要的檔案

如果只想丟最少檔案，請至少丟：

```text
NOTEBOOKLM_PROMPT.md
PROJECT_CONTEXT.md
MODEL_RESULTS.md
SOURCE_LINKS.md
```
